- **Split `src/index.ts` (603 lines) into modules** (scan/codegen/validate/
  watcher): not worth doing now. The file is cohesive and unusually
  well-commented, the pure functions are already cleanly separated and exported,
  and the test suites are already split by concern. The refactor's churn and
  export-stability risk outweigh the navigability gain at the current size.
  Revisit if the file grows materially or gains a second source file.
- **Switch build from `tsc` to a bundler (tsdown/tsup/unbuild)**: deferred.
  Plain `tsc` is genuinely fine for a single ESM-only source file; tsdown is the
  on-trend Vite-adjacent option but the benefit (bundling, dual ESM/CJS, .d.ts
  rollup) is DX-nice, not necessary. Not a correctness issue.
- **Ship CJS / change `exports` ordering**: rejected — ESM-only is correct for a
  Vite plugin in 2026, and the package's `exports` already orders `types` first.
  (A `publint`/attw CI gate to *prevent future regressions* is folded into 005.)
- **Root `tsconfig.json` has no `include`** (root `build: tsc` compiles the whole
  tree): minor. The real build is `packages/plugin`'s; the root `build` script is
  effectively unused. Not worth a dedicated plan — fix opportunistically.

## Architecture edge cases noted, not planned (low value at current scale)

- **Multiple plugin instances collide on the virtual module ID** (`test.plan.md`
  bug #9): `VIRTUAL_MODULE_ID = 'virtual:static-assets'` is a module-level const
  (`src/index.ts:8`), so two plugin instances (e.g. scanning two directories)
  would both claim the same id — last one wins. Real limitation, but no evidence
  of demand (no issue, single-directory is the documented use case). Would need a
  configurable/instance-scoped module id. Revisit if users request multi-directory
  scanning.
- **Parallel-build dts write race** (`test.plan.md` bug #10): `writeDtsFile`
  read-compares-then-writes `typesOutputFile`; concurrent Vite builds targeting the
  same output file could race. Exotic scenario; the "only write if changed" guard
  makes identical-content races harmless. Not worth a plan now.
